The CPU Scheduler AI Simulator is a full-stack educational web application designed to visualize and analyze various CPU scheduling algorithms in real time. 
The platform allows users to simulate algorithms such as FCFS, SJF, Priority Scheduling, and Round Robin through interactive process management and animated Gantt charts.
It features a modern responsive UI built with HTML, CSS, and JavaScript, along with real-time performance analytics using Chart.js. 
The simulator also includes AI-based recommendations that help identify the most efficient scheduling algorithm for a given workload. 
Users can experiment with industrial scenario presets like banking systems, hospital emergency handling, and embedded systems. 
Authentication and simulation history management are powered by Supabase with secure database integration. 
The project was developed collaboratively by our  Team Logical Forge to simplify Operating System scheduling concepts through practical visualization and interactive learning.
